# WSM groups as teams

WSM is the only series here that runs groups before the final. These pages pool every athlete from every group into one virtual comp, scored under WSM Linear using raw event data (actual times, reps, distances). Per-group total = sum of its members' points. Addresses claims that some groups were stacked harder than others.
